  • open popup by clicking on button next to search input in upper left corner

  • click on plus/minus sign to change zoom level separately for:
    - list of messages
    - received messages
    - composed messages
  • click on zoom percentage to reset it to 100%

    link to old video, prior to version 2.0
  • zoom is saved, no need to set it every time you open Gmail
  • zoom range is from 30-250%
  • it doesn't affect the actual font size, and doesn't break any layouts (eg. styled HTML mails)
  • If you use zoom for for composed messages, your recipients will receive it with normal font size, or whatever size you choose from edit toolbar
  • This extension does not use any additional resources in everyday work (reading mails, changing zoom...)
  • Only time you'll see it in Chrome Task Manager (Shift+Esc) is when this options (info) page is active
  • 10-15 sec after you close this page or navigate away, memory is released
  • If you like it, star it. Links are on the left side of the screen.
